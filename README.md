# Camp Ezekiel 2026

Official website for **Camp Ezekiel** — hosted by Forward In Faith Ministries International, USA Youth & Young Adults, South Region.

One camp, two tracks: **Youth** and **Young Adults**.

- **Dates:** July 31 – August 2, 2026
- **Location:** Camp Victory — 2953 Willafa Woods Rd, Cartwright, OK 74731
- **Cost:** $150 per person
- **Coordination:** +1 (469) 882-6878

## Viewing the site

The site is a static page (`index.html` + `logo.png`) — just open `index.html` in any browser.

## Hosting

The site is served at **https://southregionyouth.org** via Cloudflare Pages (connected to this repo — every push to `main` redeploys automatically).

It is also published with **GitHub Pages** at `https://kaizokuzard-hash.github.io/camp-ezekiel/`.
